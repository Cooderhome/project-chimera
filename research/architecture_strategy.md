# Project Chimera: Architecture & Strategy

## 1. Architectural Patterns
- FastRender Swarm: Planner → Worker → Judge
- Model Context Protocol (MCP) for external API standardization

## 2. Agentic Commerce
- Non-custodial crypto wallets
- Autonomous transactions via Coinbase AgentKit

## 3. Operational Model
- Single human Super-Orchestrator
- Self-healing workflows
- Centralized context management (BoardKit)

## 4. Technology Stack
- LLMs: Gemini 3 Pro / Claude Opus
- Databases: Weaviate, PostgreSQL, Redis
- Cloud: AWS/GCP with Kubernetes
- MCP Servers for integration
